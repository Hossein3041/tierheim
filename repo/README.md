# tierheim

Im Rahmen des Praxissemesters wurde an einer webbasierten Tierheim-Management-Anwendung
zur digitalen Verwaltung von Tierdaten, Terminen und administrativen Prozessen gearbeitet.
Der Schwerpunkt lag auf der Full-Stack-Entwicklung mit klarer Trennung von
Backend- und Frontend-Komponenten.
Im Backend wurden RESTful Webservices mit Java (Micronaut) umgesetzt,
einschließlich eines schichtenbasierten Architekturmodells mit DTOs und der
Anbindung einer PostgreSQL-Datenbank.
Im Frontend wurde eine Single Page Application mit React / Next.js auf Basis
komponentenbasierter Entwicklung realisiert.
Die Entwicklung erfolgte im Team unter Einsatz von Git zur Versionsverwaltung.

Das Praktikumsbericht ist in Bearbeitung.
